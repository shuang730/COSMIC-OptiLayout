COSMIC OptiLayout is a modern optical layout design and 2D ray-tracing tool developed in MATLAB for research laboratories.

Download Latest Version View on GitHub

Designed by researchers, for researchers.

COSMIC OptiLayout focuses on fast optical layout design, real-time ray tracing, flexible component editing, measurement, alignment, and figure export. It is made for lab planning, optical discussions, teaching, reports, and presentations.

Features

⚡

Real-time 2D Ray Tracing

Visualize optical paths instantly while editing light sources and optical components.

💡

Parallel & Point Sources

Adjust beam width, angle, ray count, emission angles, and ray colors.

🔬

Optical Components

Lens, Objective, Mirror, Beam Splitter, Pinhole, Camera, and Surface.

🎛️

Editable Parameters

Modify position, rotation, diameter, focal length, NA, aperture, and more.

📏

Measurement Tools

Measure point-to-point and surface-to-surface distances directly in the layout.

🎯

Alignment Tools

Automatically align selected components along the X or Y direction.

🖼️

PNG / PDF Export

Export clean optical layout figures for reports, slides, and documentation.

↩️

Productivity Workflow

Save/load layouts, undo/redo, box selection, zoom, pan, and view lock.

Software Preview

Build and edit optical systems visually with a practical interface designed for experimental optics.

Applications

COSMIC OptiLayout is designed for optical research, teaching, and experimental planning.

Optical Layout Design

Quickly sketch and test experimental optical paths.

Beam Alignment

Plan laser paths, mirrors, beam splitters, and detection geometry.

Microscope Systems

Represent objectives, imaging paths, cameras, and apertures.

Teaching

Use interactive ray tracing to explain geometric optics concepts.

macOS Installation Note

If macOS reports that MyAppInstaller_mcr.app is damaged and cannot be opened, this is caused by macOS Gatekeeper. The installer is not actually damaged.

Open Terminal and run:

xattr -dr com.apple.quarantine

Then drag MyAppInstaller_mcr.app into the Terminal window to insert its path, press Enter, and launch the installer again.
